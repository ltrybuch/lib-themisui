Dos

  • Reuse helper classes and/or variables when possible.
  • When a rare or non-reusable spacing pattern is required (e.g. 64px) encapsulate those within a component’s or page’s styles.

Don'ts

  • Don’t define spacing on global elements (p, h1, h2, etc).

Spacing helpers

There are 3 types of helpers used for building spacing in layouts.

  1. Insets: used for adding padding to all four sides of an element.
  2. Stacks: used for adding bottom margin to an element for vertical spacing.
  3. Inlines: used for adding right margin to an element for horizontal spacing.
